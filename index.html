Tourism Site Recommendation Project

Project Overview

This project aims to recommend tourism destinations in Rwanda based on user interests, such as wildlife, culture, adventure, relaxation, and eco-tourism.

Key Features

  • Personalized destination suggestions.
  • Simple decision tree logic for recommendations.
  • Focus on Rwanda's diverse tourist attractions.
